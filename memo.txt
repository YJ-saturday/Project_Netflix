components: UI를 담당하는 React 컴포넌트가 들어감.

utils: 비주얼과 관계없는 기능(로직, 헬퍼 함수, 데이터 처리 등)을 모아두는 곳.
/src
 ├── utils/
 │   ├── formatDate.js      → 날짜 포맷 변환 함수
 │   ├── api.js             → API 요청 함수 (데이터 가져오기/보내기)
 │   ├── validators.js      → 입력값 검증(Validation) 함수
 │   ├── localStorage.js    → 로컬 스토리지 저장/불러오기 함수
 │   ├── helpers.js         → 기타 공통적으로 사용되는 함수


GET: 서버에서 데이터를 가져올 때 사용 (읽기)
- 서버에서 데이터를 가져올 때(GET) 사용.
- URL의 쿼리 스트링(?key=value)을 통해 데이터를 전송.
- 데이터를 변경하지 않고, 조회만 함 (안전한 요청).

POST: 서버에 데이터를 생성할 때 사용 (쓰기)
- 서버에 데이터를 보낼 때(생성, 저장, 업데이트) 사용.
- 데이터를 body에 담아서 전송 (URL에 노출되지 않음).
- 로그인, 회원가입, 데이터 저장 등의 요청에서 사용됨.

PUT/PATCH: 데이터를 업데이트할 때 사용
DELETE: 데이터를 삭제할 때 사용
